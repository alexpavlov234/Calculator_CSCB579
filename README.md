# Windows Forms Калкулатор

Разработен от Александър Александров Павлов F112657

## Описание на проекта
Просто приложение за калкулатор, създадено с C# и Windows Forms.

## Функционалности
*   **Основни аритметични операции:** Събиране, изваждане, умножение и деление.
*   **Десетични числа:** Пълна поддръжка за изчисления с десетична запетая.
*   **Промяна на знака на числото:** Лесно превключване между положителни и отрицателни числа.
*   **Контрол на входа:**
    *   **Изчистване (C):** Нулира цялото изчисление.
    *   **Изтриване (→):** Работи като "Backspace" за премахване на последния въведен символ.
*   **Обработка на грешки:** Показва съобщения за грешки при невалидни операции, като деление на нула.

## Как да стартирате проекта
1.  Клонирайте хранилището на вашата локална машина.
2.  Отворете файла на решението (`.sln`) в JetBrains Rider или Visual Studio. Бъдете сигурни, че имате инсталирани .NET 8 SDK и Windows Forms компоненти.
3.  Стартирайте проекта.

## Структура на кода
*   `Form1.cs`: Съдържа цялата логика за калкулатора. Обработва кликванията на бутоните, управлява състоянието (текущи числа, избрана операция) и извършва изчисленията.
*   `Form1.Designer.cs`: Автоматично генериран файл от дизайнера на Windows Forms. Съдържа кода за инициализиране и подреждане на всички UI компоненти.
*   `Operation.cs`: Прост `enum`, който дефинира възможните аритметични операции за по-добра четимост на кода.
